# Todo App

A simple task management web application built with PHP and MySQL. Users can register, log in, create and manage tasks, organize them into projects, upload attachments, and export or import task data. The application also includes an admin panel for user management and activity logs.

> **AI-assisted project:** This project was built with significant assistance from AI tools. A large portion of the code was generated with AI support, while I focused on integrating the components, understanding the implementation, debugging issues, and ensuring the application worked correctly as a complete system.

## Features

- User registration and authentication
- Create, edit and delete tasks
- Organize tasks into projects
- Task priorities and due dates
- File attachments for tasks
- Import tasks from JSON
- Export tasks to CSV, JSON and XML
- User settings management
- Admin panel with user management and activity logs

## Technologies

- PHP
- MySQL
- HTML
- CSS
- XAMPP (Apache + MySQL)
- phpMyAdmin

## Setup

1. Install XAMPP.
2. Clone or download the repository into the `htdocs` directory of your XAMPP installation.

   Example:

   ```
   C:\xampp\htdocs\todo-app
   ```

3. Start **Apache** and **MySQL** from the XAMPP Control Panel.
4. Import the provided SQL file into phpMyAdmin.
5. Open the application in your browser:

   ```
   http://localhost/todo-app/public/register.php
   ```

6. Register a new account and log in to access the application.
